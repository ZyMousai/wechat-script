# wechat-script
##开发环境
1. python3.7.3 32bit
2. 依赖包 requirements.txt
3. 安装依赖 pip install -r requirements.txt

### 目录说明
#### core 核心代码
1. callback_fun.py 主要功能回调函数
2. mongo_client mongo封装
3. pubilc_fun 公共方法
4. wxcom.py 调用dll
#### tools 工具包
1. auto_recv 自动回复使用到的工具
2. dll
3. im_file 导入自动添加好友使用
#### env 全局
#### auto_recv_manage 自动回复入口
